# 小红书内容策略 v2.0（2026-05-06 战略转折）

## 核心认知转向

**旧方向（已放弃）：** AI生成虚拟笔记 → 没有真实经历支撑 → 数据惨淡

**新方向：** AI当剪辑师，大威提供真实素材 → 窑滚人生漫画IP + 真实手作体验

## 关键教训

### 为什么AI笔记不行
- 六篇10万+赞爆款分析：全都是**纪录片式**真实内容（DIY过程、价格透明、翻车、店名互动）
- 爆款DNA = 视觉钩子 + 价格争论点 + 工艺纠错空间 + 真人店名
- 评论区才是本体，AI生成的内容留不住人

### 为什么转向漫画
- 墨家军核心竞争力：AI生图 + GPT Image 2 + PIL中文气泡
- 四格漫画天然适合小红书图文形式
- 蓝海赛道：陶瓷拟人化+搞笑=零竞品

## 新内容生产线

```
真实素材（大威） → 墨家军包装 → 发布
    ├── 拉坯手绘过程 → 漫画脚本 + 出图 + 气泡
    ├── 出窑对比 → 分两篇发（蹲后续+成品）
    └── 价格账单 → 评论区引爆点
```

## 数据驱动策略

### 采集管道
- 小墨(Coze) → 邮件 → 小云(CORE-01) → xhs_hotspot_data表
- 三类数据：漫画竞品/职场女性热点/景德镇流量密码
- 频率：每天凌晨1点

### 反馈闭环
采集 → 分析爆款DNA → 更新Few-shot锚点 → 优化漫画 → 发布 → 看数据 → 再采集

## 内容测试原则
- 先发再多想：EP1效果不好就改，不是先想完美再发
- 评论区挖需求：评论里问什么就下期做什么
- 每周至少2期保持存在感

## 1. 用户研究框架

### 小红书"水源地"研究法
天青浅目标用户活跃在这5类水源地：

| 水源地 | 做什么 | 怎么挖 |
|--------|--------|--------|
| 小红书评论区 | 竞品笔记下用户在讨论什么 | 搜"景德镇陶瓷""手工杯"，读高赞评论 |
| 淘宝/京东评价 | 买类似产品的人怎么评价 | 搜"手绘茶杯"，读好评+差评 |
| 抖音陶瓷话题 | 短视频用户在说什么 | 搜"景德镇""陶瓷杯"，看弹幕和评论 |
| 微信朋友圈 | 目标人群晒的生活方式 | 关注"茶具""茶道""手作"相关晒单 |
| 线下市集/展会 | 面对面接触真实用户 | 观察谁拿起松鼠杯、看了多久、问了什么 |

### 提取框架（每条评论/对话提取）
1. 用户怎么描述她的问题？（原话，不翻译）
2. 她目前在用什么替代方案？
3. 她为什么还没买？（价格/审美/信任/无需求）
4. 什么样的描述让她"心动"了？

## 2. 转化文案原则

### 天青浅文案铁律
1. **利益 > 特征**：不说"手绘青花"，说"每只杯子都不一样，只属于你"
2. **具体 > 抽象**：不说"高品质"，说"画一只杯子要3天，烧坏6只才出1只成品"
3. **用户语言 > 行业术语**：不说"釉下彩"，说"怎么洗都不掉色"
4. **一个画面 > 十行文字**：让用户脑中自动出图

### 小红书笔记转化结构
```
标题（造需求）→ 正文（建信任）→ 结尾（给行动）→ 图片（封神）
```

### 三种信任钩子
| 钩子 | 例句 |
|------|------|
| 工艺钩子 | "画了12年瓷器的师傅，一天只能画3只" |
| 产地钩子 | "602所长大的景德镇人告诉你" |
| 使用钩子 | "用了3个月之后，每天早上第一件事就是用这个杯子" |

## 3. 定价策略

### 松鼠杯定价诊断
**当前价格**：598-798元

**价值指标（用户用什么衡量"值不值"）**：
- 手绘工时（"这只杯子花了多少时间"）
- 独特性（"世界上只有这一只"）
- 使用频率（"每天都会用到"）

**好坏优三级结构建议**：

| 级别 | 产品 | 价格 | 定位 |
|------|------|------|------|
| 好 | 基础款松鼠杯 | 298-398 | 试水入门 |
| 优 | 手绘定制款 | 598-798 | 主力利润 |
| 典藏 | 大师亲绘限量 | 1298-1698 | 锚定高端 |

**涨价信号**：如果手绘订单排到2周以上 → 该涨了。

### Van Westendorp定价测试法
问用户4个问题：
1. 什么价格你觉得太便宜会怀疑质量？（太低）
2. 什么价格你觉得划算？（便宜）
3. 什么价格你觉得开始贵了？（贵）
4. 什么价格你觉得太贵不会买？（太贵）

画曲线交点 = 最优价格区间。

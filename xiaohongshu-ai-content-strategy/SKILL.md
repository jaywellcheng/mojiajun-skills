# 小红书 AI 内容策略核心认知

## 一句话总结
AI写不出真实内容。墨家军的正确角色是**大威的剪辑师**，不是**替代大威的写手**。

## 7篇10万+赞爆款DNA分析（2026-05-06）

分析了7篇景德镇陶瓷类爆款笔记（8.4万~89.2万赞），发现核心规律：

**爆款不是"写得好"，是"有东西可吵"**。每篇评论区都比正文精彩。

### 六引擎模型

| 引擎 | 机制 | 示例 |
|------|------|------|
| 视觉钩子 | 成品图要有"信息差"——不是美，是"怎么做到的""我也想要" | 烟花碗透光、花瓶画了10小时 |
| 价格透明 | 评论区第一高楼永远是价格讨论 | 835元30个→100+楼争论 |
| 技术争议 | 故意留"可纠正的点"，让内行来纠错 | "别拿把手！"322赞 |
| 店铺饥渴 | 永远有人问"哪家店" | "一丛家的窑"出现4篇 |
| 蹲后续 | 出窑前一篇+出窑后一篇，两篇都爆 | "蹲"字是小红书最强互动词 |
| 过程>结果 | 画废的5个+成功的1个 > 直接晒成品 | 翻车比完美更真实 |

### 评论区自生长模式
- 价格撕：100+楼争论→持续推送
- 科普纠错：内行帮外行→二次传播
- 同款分享：社区验证→信任背书
- @朋友：社交裂变引擎
- 蹲踢："出了踢我"→订阅式关注

### 标题三大公式
- 悬念型："画了X小时 + 结果竟然/老板都/我自己不敢认"
- 对比型："素坯 vs 成品 / 以为 vs 实际"
- 身份型："到了景德镇像掉进米缸的老鼠"

## 正确分工

```
大威负责：经历 + 素材
  ├─ 拉坯、手绘、出窑 → 拍照/录视频
  ├─ 记价格：素坯X + 驻场费X + 烧制X = 总价X
  ├─ 翻车了 → 拍照记录
  └─ 真实感受 → 口述或记两句话

墨家军负责：包装 + 分发
  ├─ 把照片整理成图文结构
  ├─ 算账、做价格透明度
  ├─ 写标题（悬念/对比/身份型公式）
  ├─ 加标签 + 互动引导
  └─ 发布 + 数据追踪
```

## 爆款公式

```
爆款 = 真实素材 + 价格透明 + 争议切口 + 店铺关键词
```

## 不再做的事

- ❌ AI从头写笔记（假内容，读不进去）
- ❌ 用AI示例库做锚点（AI学AI，越来越平）
- ❌ 试图用Prompt工程弥补素材缺失

## 墨家军管线调整

旧流程：选题 → AI生成正文 → 自审 → 发布
新流程：大威给素材 → AI结构化 → 大威审 → 发布

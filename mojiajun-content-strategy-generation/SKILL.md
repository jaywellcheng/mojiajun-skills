# 内容策略生成工作流

## 概述

输入：品牌/产品信息 + 标题公式库 → 输出：可执行的内容策略文档。

适用于：
- 新品牌小红书冷启动
- 新产品上市内容规划
- 已有品牌的内容策略迭代

## 输入要素

1. **品牌信息**：名称、创始人身份、差异化点
2. **产品信息**：名称、参数、定价、限量/常规
3. **目标用户**：年龄、性别、消费力、兴趣
4. **标题公式库**：从 `mojiajun-xiaohongshu` 技能加载

## 输出结构

```
一、品牌核心定位（差异点提炼）
二、标题策略（3个主攻公式 + 候选标题）
三、内容日历模板（7天轮转）
四、四阶段策略（相识→相知→相爱→在一起）
五、单篇笔记SOP
六、对标参考
七、关键数字
```

## 标题匹配规则

| 品牌类型 | 主攻公式 | 理由 |
|---------|---------|------|
| 有真实创始人故事 | 公式10「我即产品」 | 身份=最强信任资产 |
| 限量/手工产品 | 公式5「稀缺独享」 | 天然稀缺性 |
| 地域特色产品 | 公式9「本地人设」 | 地域=天然信任背书 |
| 攻略/旅游类 | 公式1+2+6 | 身份反转+焦虑+攻略痛点 |
| 美食推荐类 | 公式3+4 | 数字清单+情绪反差 |

## 候选标题生成规则

- 每个公式至少生成3个候选标题
- 每个标题触发≥2个心理开关
- 用5问自检清单过滤

## 内容日历模板

```
周一 文化 → 深度内容（工艺/匠人故事）
周二 攻略 → 涨粉内容（旅游/本地推荐）
周三 产品 → ★产品种草日
周四 美食 → 引流内容（美食/生活）
周五 生活 → 人设内容（日常/故事）
周六 产品 → ★第二个产品日
周日 攻略 → 合集/复盘
```

## 使用示例

```python
# 输入
brand = {"name": "天青浅", "founder": "大威，45岁景德镇人定居深圳"}
product = {"name": "松鼠杯", "price": "598-798元", "quantity": "50只限量"}

# 输出
strategy = generate_strategy(brand, product)
# → 主攻公式10「我即产品」+ 辅助公式5「稀缺独享」
# → 候选标题：*"景德镇人的松鼠杯快做好了"*
# → 内容日历：周三/周六陶瓷产品日
```
